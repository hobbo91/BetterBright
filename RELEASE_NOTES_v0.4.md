# BetterBright v0.4

## Release files

- `BetterBright_v0.4.zip` - end-user release containing:
  - `BetterBright.ini`
  - `BetterBright.prx`
- `BetterBright_source.zip` - source archive containing the contents of `src/`.

## Install

Copy `BetterBright.prx` and `BetterBright.ini` together into your PSP plugins folder, then enable BetterBright from ARK-4 / FasterARK.
